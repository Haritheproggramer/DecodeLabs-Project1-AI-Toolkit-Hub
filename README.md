# Decode Labs Project 1 – AI Toolkit Hub

This project was developed as part of the Decode Labs Internship Program (Project 1).

AI Toolkit Hub is a multi-page frontend project built with HTML5, CSS3, and vanilla JavaScript. It helps students and developers discover useful AI tools for learning, coding, productivity, research, creativity, and audio work.


## Features

- Semantic HTML structure
- Mobile-first responsive design
- Dark theme with consistent spacing and readable contrast
- Multi-page navigation with active states
- Tool directory with categorized sections
- Tool detail pages with practical summaries
- About section with project purpose
- Contact form validation with success feedback
- Mobile navigation toggle
- Safe external links with `noopener noreferrer`

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run the Project Locally

1. Download or clone the repository.
2. Open the `AI-Toolkit-Hub` folder in VS Code or any code editor.
3. Open `index.html` in a browser.
4. Resize the browser to test the responsive layout.
5. Navigate through the pages and tool detail views to verify the shared layout.

## GitHub Pages Deployment Steps

1. Push the project to GitHub.
2. Go to the repository **Settings** → **Pages**.
3. Set **Source** to `Deploy from a branch`.
4. Select the **Branch**: `main`.
5. Select the **Folder**: `/ (root)`.
6. Save and wait for GitHub Pages to build.
7. Open the live site URL shown in the Pages section.


## Vercel Deployment Steps

1. Import your GitHub repository into Vercel.
2. Ensure the **Production Branch** is set to `main`.
3. Trigger deployment using the latest commit on `main`.
4. If auto-deploy is enabled, pushing to `main` will redeploy automatically.
